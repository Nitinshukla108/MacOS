# 🍎 MacOS Portfolio UI (React)

An interactive **MacOS-inspired portfolio** built using React.  
This project simulates a desktop environment with a **dock, windows, and CLI terminal**, allowing users to explore my work in a unique way.

---

## 🚀 Features

- 🖥️ MacOS-style Desktop Interface
- 📂 Window System (Open / Close / Resize)
- 💻 Interactive CLI (Command-based navigation)
- 🎯 Animated Dock (Mac-style hover effect)
- 📄 Resume Viewer (PDF support)
- 📝 Notes App (Text file preview)
- 🎧 Spotify UI Integration
- 🔗 GitHub & External Links
- 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

- ⚛️ React.js
- 🎨 SCSS (Sass)
- ⚡ Vite
- 💻 react-console-emulator
